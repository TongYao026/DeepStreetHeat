# DeepStreetHeat Web Demo

Single-page GIS demo for the course deliverable emphasis: User Interface Design, Data Flow, and System Design.

## Streamlit Deployment (Recommended for Presentation)

From the project root:

1.  Start a local file server for images (required for the interactive map):
    ```powershell
    python -m http.server 8000
    ```
2.  Run the Streamlit app:
    ```powershell
    streamlit run streamlit_app.py
    ```

The Streamlit app provides a professional interface with project metadata, model results, and the embedded interactive GIS explorer.
